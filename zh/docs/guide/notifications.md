# 通知中心

RDE 内置通知系统，支持站内消息和多渠道外部推送。

## 站内通知

- **未读/已读管理**：通知标记已读、批量标记
- **分类筛选**：按通知类型（系统、备份、下载、安全）筛选
- **通知列表**：时间线形式展示所有通知
- **桌面通知**：任务栏托盘区域实时显示未读数

## 推送渠道

支持将重要通知推送到外部渠道：

| 渠道 | 说明 |
|------|------|
| 邮件 | SMTP 邮件通知 |
| Telegram | Telegram Bot 推送 |
| Bark | iOS Bark 推送 |
| Webhook | 自定义 HTTP 回调 |
| 钉钉 | 钉钉机器人 |
| 微信 | 企业微信机器人 |

## 推送规则

可为不同事件类型配置不同的推送渠道：

- 备份完成/失败 → 邮件 + Telegram
- 磁盘空间不足 → 所有渠道
- 下载完成 → Bark
- 安全告警 → 邮件

## 免打扰

支持设置免打扰时段，在指定时间内不发送外部推送通知。
