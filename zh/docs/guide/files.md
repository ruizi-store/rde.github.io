# 文件管理

RDE 内置功能完整的文件管理器，在浏览器中提供桌面级的文件操作体验。

## 基本操作

- **浏览**：树形目录导航，支持面包屑路径栏
- **新建**：创建文件、文件夹
- **复制/剪切/粘贴**：支持快捷键操作
- **重命名**：双击文件名或 F2 快捷键
- **删除**：支持回收站和直接删除
- **搜索**：按文件名搜索

## 文件上传

支持多种上传方式：

- **拖拽上传**：将文件拖拽到文件管理器窗口
- **点击上传**：通过上传按钮选择文件
- **文件夹上传**：支持整个文件夹上传

### 大文件断点续传

基于 TUS 协议实现断点续传，支持 10GB+ 大文件上传：

- 网络中断自动恢复
- 显示上传进度
- 支持暂停/恢复

## 文件下载

- 单文件直接下载
- 多文件/文件夹打包为 ZIP 下载

## 文件预览

内置多种文件预览器：

| 类型 | 支持格式 |
|------|---------|
| 图片 | JPG, PNG, GIF, SVG, WebP |
| 视频 | MP4, WebM, MKV |
| 音频 | MP3, FLAC, WAV, OGG |
| 文档 | TXT, Markdown, PDF |
| 代码 | 支持语法高亮的代码查看器 |

## 文件分享

支持生成分享链接，方便将文件分享给他人：

- 设置分享密码（可选）
- 设置过期时间
- 支持限制下载次数

## Samba 共享

一键配置局域网文件共享服务，让其他设备通过 SMB 协议访问 RDE 上的文件。
