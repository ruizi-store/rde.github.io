# 备份与恢复

RDE 提供系统配置和用户数据的备份、还原、迁移功能，支持云服务器和 NAS 私有部署场景。

## 系统配置备份

| 备份项 | 说明 |
|--------|------|
| RDE 应用配置 | 所有模块的设置、偏好 |
| 用户账户 | 用户信息、权限、SSH 密钥 |
| Docker 容器配置 | docker-compose.yml、环境变量（不含镜像） |
| 网络配置 | DDNS、反向代理、端口映射规则 |
| 计划任务 | 定时任务、自动化脚本 |
| Samba 共享配置 | 共享目录、访问权限 |

## 用户数据备份

| 类型 | 策略 |
|------|------|
| 文件/文档 | 全量 + 增量备份 |
| 照片库 | 含元数据、相册结构 |
| 数据库 | SQLite 导出 |
| 应用数据目录 | 按应用选择性备份 |

## 备份目标

支持多种备份目标：

- **本地磁盘**：备份到本地其他分区或外置硬盘
- **WebDAV**：坚果云、ownCloud 等
- **S3 兼容**：AWS S3、MinIO、腾讯云 COS
- **SFTP**：通过 SSH 协议备份到远程服务器
- **Rclone**：支持 40+ 云存储服务

## 备份策略

- **手动备份**：随时手动触发
- **定时备份**：设置 cron 计划
- **增量备份**：仅备份变更文件，节省空间
- **备份保留策略**：自动清理过期备份

## 恢复

- 从备份文件恢复系统配置
- 选择性恢复特定模块的数据
- 跨设备迁移

::: tip 云备份
会员用户还可使用 [云备份](/zh/docs/guide/cloud-backup) 服务，自动备份到云端。
:::
