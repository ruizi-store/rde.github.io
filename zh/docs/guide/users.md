# 用户管理

RDE 支持多用户并发使用，每个用户拥有独立的桌面空间。

## 用户模型

- 所有用户均具有 admin/sudo 权限
- 支持 2 人并发在线
- 每个用户对应一个自动创建的 Linux 系统用户

## 用户隔离

| 资源 | 隔离方式 |
|------|---------|
| 文件系统 | 每个用户有独立的 home 目录 |
| 终端会话 | 按用户隔离的 PTY 会话 |
| 桌面空间 | 独立的窗口布局和壁纸 |
| Docker | 共享（所有用户可管理容器） |

## 用户操作

### 创建用户

管理员可在设置中添加新用户：

1. 打开「系统设置」→「用户管理」
2. 点击「添加用户」
3. 设置用户名和密码
4. RDE 自动创建对应的 Linux 系统用户

### 删除用户

删除用户时可选择是否保留用户数据。

## 认证方式

- 用户名 + 密码登录
- 支持 JWT Token 认证
- 会话管理和自动过期
