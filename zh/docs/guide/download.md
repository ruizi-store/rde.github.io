# 下载管理

RDE 内置基于 Aria2 的多协议下载管理器，支持 WebSocket 实时进度推送。

## 支持的协议

- **HTTP/HTTPS**：直链下载
- **BT 种子**：上传 .torrent 文件或粘贴磁力链接
- **磁力链接**：`magnet:?xt=...` 格式

## 功能特性

- **多线程下载**：自动分段并行下载，加速大文件
- **断点续传**：下载中断后自动恢复
- **实时进度**：WebSocket 推送下载速度和进度，无需刷新页面
- **下载队列**：支持批量添加下载任务
- **下载历史**：记录所有下载任务，支持重新下载

## 使用方式

1. 点击任务栏下载图标或通过开始菜单打开
2. 粘贴 URL 或磁力链接
3. 选择保存路径
4. 开始下载

## 下载统计

查看下载数据统计：

- 累计下载量
- 下载速度趋势
- 任务完成情况
