更多应用 ​

除了核心的文件管理、Docker、终端外，RDE 还内置了丰富的桌面应用。

🎵 音乐播放器 ​

功能完整的桌面音乐应用：

  • 多种显示模式：迷你模式、窗口模式、壁纸模式
  • 歌词显示：支持 LRC 歌词同步显示
  • 音频可视化：频谱柱、环形、万花筒等多种可视化效果
  • 播放列表：创建和管理多个播放列表
  • 支持格式：MP3, FLAC, WAV, OGG, AAC 等

🎬 视频播放器 ​

多格式桌面视频播放器：

  • 广泛格式支持：MP4, MKV, AVI, WebM 等
  • 实时转码：ffmpeg HLS 转码，浏览器不支持的格式也能播放
  • 字幕支持：自动加载同名字幕文件
  • 播放记忆：记住上次播放位置
  • 倍速播放：0.5x ~ 3x 播放速度
  • 画中画：浮窗模式播放
  • 文件管理器集成：双击视频文件直接播放

📸 相册 ​

类群晖/Immich 的索引式相册系统：

  • 自动索引：扫描指定目录的照片
  • 元数据提取：EXIF/GPS 信息读取
  • 缩略图生成：自动生成多尺寸缩略图
  • 相册管理：创建和整理相册
  • 时间线视图：按日期浏览照片

🎮 复古游戏 ​

基于 EmulatorJS（WebAssembly）的纯前端复古游戏模拟器：

  • 支持平台：NES, SNES, GBA, GBC, N64, PS1, NDS, Sega MD 等 12 个经典平台
  • ROM 管理：上传和管理游戏 ROM 文件
  • 手柄支持：连接 USB/蓝牙手柄游玩
  • 存档管理：随时存档/读档
  • 纯前端运行：无需额外后端服务

📦 Flatpak 应用 ​

在浏览器中运行 Linux 桌面 GUI 应用：

  • 通过 KasmVNC 实现浏览器内 GUI 渲染
  • Flatpak 作为应用来源，跨发行版通用
  • 支持多会话并发
  • 应用安装/卸载管理

📊 系统监控 ​

实时系统监控仪表盘：

  • CPU 使用率和负载
  • 内存使用情况
  • 磁盘空间和 I/O
  • 网络流量
