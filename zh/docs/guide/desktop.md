# 桌面环境

RDE 提供类 Windows 的 Web 桌面体验，在浏览器中即可获得完整的桌面操作环境。

## 窗口管理器

窗口管理器是 RDE 桌面的核心组件，管理所有应用窗口的生命周期、布局和交互。

### 窗口操作

- **拖拽移动**：按住标题栏拖拽窗口到任意位置
- **缩放大小**：拖拽窗口边缘调整大小
- **最大化**：双击标题栏或点击最大化按钮
- **最小化**：点击最小化按钮，窗口收纳到任务栏
- **关闭**：点击关闭按钮或使用快捷键

### 窗口类型

RDE 支持三种窗口类型：

| 类型 | 说明 | 适用场景 |
|------|------|---------|
| 组件窗口 | Svelte 组件直接渲染 | 文件管理器、设置等原生应用 |
| iframe 窗口 | 嵌入外部页面 | Web 应用、第三方服务 |
| 弹窗 | 模态/非模态对话框 | 确认框、属性面板 |

### 多窗口管理

- 支持多窗口同时打开，自动管理层叠顺序（z-index）
- 点击窗口自动置顶
- 窗口间支持拖拽传递数据

## 任务栏

屏幕底部的任务栏提供：

- **应用图标**：常用应用快捷入口
- **已打开窗口**：点击切换或最小化/恢复窗口
- **系统托盘**：时间、通知、网络状态等
- **开始菜单**：所有已安装应用列表

## 右键菜单

桌面和文件管理器支持右键菜单操作，提供：

- 新建文件/文件夹
- 粘贴、刷新
- 更换壁纸
- 显示设置

## 多用户支持

RDE 支持多用户并发使用：

- 每个用户拥有独立的桌面空间
- 文件系统按用户隔离
- 终端会话按用户隔离
- Linux 系统用户自动创建

详见 [用户管理](/zh/docs/guide/users)。
