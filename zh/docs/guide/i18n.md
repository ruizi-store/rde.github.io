# 国际化与镜像加速

RDE 支持多语言界面，并为中文用户提供一键国内镜像源加速功能。

## 语言支持

目前支持：

- 🇨🇳 简体中文 (`zh-CN`)
- 🇺🇸 English (`en-US`)

在「系统设置」→「语言」中切换。

## 国内镜像源加速

选择中文时，RDE 自动推荐并一键启用国内镜像源，覆盖以下软件源：

| 软件源 | 说明 |
|--------|------|
| APT | Debian/Ubuntu 包管理 |
| Docker | Docker Hub 镜像 |
| NPM | Node.js 包管理 |
| PIP | Python 包管理 |
| Flatpak | Flatpak 应用仓库 |
| Go | Go Modules 代理 |
| Cargo | Rust 包管理 |
| GitHub | GitHub 文件加速 |
| Maven | Java 依赖管理 |

## 配置方式

### 总开关

- **启用**：一键切换所有软件源为国内镜像
- **禁用**：恢复为官方源

### 逐一自定义

展开后可对每个软件源独立设置：

- 选择具体的镜像提供商（阿里云、腾讯云、清华 TUNA 等）
- 或使用自定义镜像地址

### 首次运行向导

初始化向导的最后一步会提示配置镜像源，后续可在设置中随时更改。
