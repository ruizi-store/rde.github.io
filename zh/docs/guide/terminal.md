# Web 终端

RDE 提供浏览器内完整的 Linux 终端体验，基于 xterm.js + WebSocket + PTY 实现。

## 功能特性

- **完整 Shell**：支持 bash/zsh 等 Shell，与 SSH 登录体验一致
- **多会话管理**：同时打开多个终端标签页
- **窗口自适应**：终端窗口大小变化时自动调整行列数
- **主题切换**：内置多种终端配色方案
- **复制粘贴**：支持文本选择和粘贴

## 使用方式

1. 点击任务栏的终端图标
2. 或通过开始菜单打开「终端」
3. 右键菜单也可选择「在此打开终端」

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+Shift+C | 复制选中文本 |
| Ctrl+Shift+V | 粘贴 |
| Ctrl+Shift+T | 新建标签页 |

## 多用户隔离

每个用户的终端会话使用独立的 Linux 系统用户，确保：

- 进程隔离
- 文件系统权限隔离
- 环境变量独立
