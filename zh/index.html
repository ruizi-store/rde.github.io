为什么选择 RDE？

不只是面板，是真正的云端桌面操作系统

原生桌面级体验

告别传统面板的表单操作。RDE 提供类 Windows 的完整桌面环境，让 Linux 管理变得直觉化。

  • ✦ 窗口拖拽、缩放、最大化、多窗口并排
  • ✦ 任务栏、右键菜单、桌面快捷方式
  • ✦ 响应式设计，手机 / 平板 / 桌面全适配

强大的文件管理

媲美本地文件管理器的 Web 体验，大文件传输也毫不费力。

  • ✦ 拖拽上传、批量操作、在线预览
  • ✦ TUS 协议断点续传，支持 GB 级文件
  • ✦ 文件分享、压缩解压、权限管理

Docker 应用，一键部署

内置应用商店，不懂 Docker 命令也能轻松部署各种服务。

  • ✦ 可视化容器、镜像、网络管理
  • ✦ 预置常用应用模板，点击即用
  • ✦ 实时日志、资源监控、自动更新

开箱即用的应用生态

一个平台，满足全部需求

Web 终端

浏览器内完整 Linux 终端，多会话管理，主题自定义。

下载管理

Aria2 驱动，HTTP / BT / 磁力全协议支持，实时进度推送。

相册

智能照片管理，EXIF 提取，缩略图生成，相册分组。

音乐播放器

桌面级音乐体验，歌词显示，音频可视化，多种播放模式。

视频播放器

多格式播放，ffmpeg HLS 实时转码，字幕支持，画中画模式。

复古游戏

WebAssembly 模拟器，NES / SNES / GBA / PS1 经典重温。

Flatpak 应用

浏览器中运行 Linux GUI 应用，跨发行版通用部署。

插件系统

Unix Socket 通信，热插拔加载，按需扩展高级功能。

国际化

中英双语，一键切换国内镜像源（APT / Docker / NPM / PIP）。
