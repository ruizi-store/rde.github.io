# More Apps

Beyond the core file manager, Docker, and terminal, RDE comes with a rich set of built-in desktop applications.

## 🎵 Music Player

A full-featured desktop music application:

- **Multiple display modes**: Mini mode, window mode, wallpaper mode
- **Lyrics display**: LRC lyrics synchronized display
- **Audio visualization**: Spectrum bars, ring, kaleidoscope and more visual effects
- **Playlists**: Create and manage multiple playlists
- **Supported formats**: MP3, FLAC, WAV, OGG, AAC, etc.

## 🎬 Video Player

Multi-format desktop video player:

- **Wide format support**: MP4, MKV, AVI, WebM, etc.
- **Live transcoding**: ffmpeg HLS transcoding for unsupported browser formats
- **Subtitle support**: Auto-load matching subtitle files
- **Playback memory**: Remember last playback position
- **Speed control**: 0.5x ~ 3x playback speed
- **Picture-in-picture**: Floating window playback
- **File manager integration**: Double-click video files to play directly

## 📸 Photos

A Synology/Immich-like indexed photo album system:

- **Auto indexing**: Scan specified directories for photos
- **Metadata extraction**: EXIF/GPS information reading
- **Thumbnail generation**: Automatic multi-size thumbnail creation
- **Album management**: Create and organize albums
- **Timeline view**: Browse photos by date

## 🎮 Retro Games

A pure frontend retro game emulator based on EmulatorJS (WebAssembly):

- **Supported platforms**: NES, SNES, GBA, GBC, N64, PS1, NDS, Sega MD and 12 more classic platforms
- **ROM management**: Upload and manage game ROM files
- **Gamepad support**: Connect USB/Bluetooth gamepads
- **Save states**: Save/load at any time
- **Pure frontend**: No additional backend service required

## 📦 Flatpak Apps

Run Linux desktop GUI applications in the browser:

- Browser GUI rendering via KasmVNC
- Flatpak as the app source, universal across distributions
- Multi-session concurrent support
- App install/uninstall management

## 📊 System Monitor

Real-time system monitoring dashboard:

- CPU usage and load
- Memory usage
- Disk space and I/O
- Network traffic
