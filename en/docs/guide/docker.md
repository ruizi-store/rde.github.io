# Docker App Store

RDE provides a visual Docker container management interface and app store — deploy and manage container apps without memorizing complex commands.

## App Store

Pre-built application templates for one-click deployment:

- Databases: MySQL, PostgreSQL, Redis, MongoDB
- Web servers: Nginx, Caddy, Apache
- Dev tools: GitLab, Jenkins, Portainer
- More community apps continuously updated

## Container Management

### Container Operations

- **Create**: Create containers from app templates or custom configurations
- **Start/Stop**: One-click container state control
- **Restart**: Restart running containers
- **Delete**: Remove unwanted containers
- **Logs**: Real-time container log viewing

### Container Details

View detailed container information:

- Running status and resource usage
- Port mappings
- Mounted volumes
- Environment variables
- Network configuration

## Image Management

- View local image list
- Pull images from Docker Hub
- Delete unused images
- View image layer information

> Docker mirror acceleration: Pulling Docker Hub images can be slow in China. RDE premium members can use the built-in Docker mirror acceleration service.

## docker-compose Support

Manage apps via docker-compose.yml files:

- Upload or paste docker-compose.yml
- One-click `up` / `down`
- View compose project status
