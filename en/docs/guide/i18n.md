# Internationalization & Mirror Acceleration

RDE supports multi-language UI and provides one-click domestic mirror source acceleration for Chinese users.

## Language Support

Currently supported:

- 🇨🇳 Simplified Chinese (`zh-CN`)
- 🇺🇸 English (`en-US`)

Switch in "System Settings" → "Language".

## Domestic Mirror Source Acceleration

When Chinese is selected, RDE automatically recommends and enables domestic mirror sources covering:

| Source | Description |
|--------|-------------|
| APT | Debian/Ubuntu package management |
| Docker | Docker Hub images |
| NPM | Node.js package management |
| PIP | Python package management |
| Flatpak | Flatpak app repository |
| Go | Go Modules proxy |
| Cargo | Rust package management |
| GitHub | GitHub file acceleration |
| Maven | Java dependency management |

## Configuration

### Master Switch

- **Enable**: Switch all sources to domestic mirrors at once
- **Disable**: Restore to official sources

### Individual Customization

Expand to configure each source independently:

- Choose specific mirror provider (Alibaba Cloud, Tencent Cloud, Tsinghua TUNA, etc.)
- Or use a custom mirror URL

### First-Run Wizard

The last step of the setup wizard prompts mirror source configuration, which can be changed later in settings.
