RDE 官网 ​

RDE（Remote Development Environment）完全免费开源的远程开发环境。

本仓库是 RDE 的官网与文档站源码，基于 VitePress 构建，部署到宝塔面板。

目录结构 ​

rde-website/
├── .vitepress/                     # VitePress 配置 + 主题
├── zh/                             # 中文文档
├── en/                             # 英文文档
├── public/                         # 静态资源
├── package.json
├── Makefile                        # 构建命令
└── README.md

快速开始 ​

bash
# 首次安装环境（Node.js + 依赖）
make setup

# 启动开发服务器
make dev

构建与部署 ​

bash
# 构建静态文件
make build

# 产物目录: .vitepress/dist/
# 将 dist/ 下所有文件上传到宝塔网站根目录即可

宝塔 Nginx 建议配置 ​

nginx
# 在宝塔网站设置 → 配置文件中添加：

# gzip 压缩
gzip on;
gzip_types text/plain text/css application/json application/javascript text/xml;

# VitePress 404 页面
error_page 404 /404.html;

# 静态资源缓存
location ~* \.(js|css|png|jpg|jpeg|gif|ico|svg|woff2?)$ {
    expires 30d;
    add_header Cache-Control "public, immutable";
}
